# Generic-banking-landing-page
This is a landing page for a bank.
## Local Development
Clone the repository

```bash
  git clone https://github.com/deen257/Generic-banking-landing-page
```
```
  cd Generic-banking-landing-page
```

```sh
  npm install && npm run dev
```
